# Admin Web

Prvi web admin MVP slice za barber booking sistem.

Sadrzi:
- staff login ekran
- admin shell i navigaciju
- dashboard pregled
- day calendar ekran
- appointments table sa filterima, paginacijom i sortiranjem
- placeholder stranice za `barbers`, `services` i `settings`

Lokalni start:
- kopirati `.env.example` u `.env.local`
- `npm install`
- `npm run dev`

Podrazumevano:
- frontend radi na `http://localhost:3001`
- backend treba da radi na `http://localhost:3000`
